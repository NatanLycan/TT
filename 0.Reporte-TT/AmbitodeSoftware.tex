\section{Ambito de Software}

El sistema a desarrollar surge con el proposito de poner al alcance de cualquier usuario con un smartphone android una herramienta que facilite la obtención de medidas en el ambito que se requiera, de una manera simple y moderna.

El producto consistira de una aplicación que permita obtener información necesaria a través de sus sensores y utilizando algoritmos necesarios para procesarla además de interactuar con el usuario para definir el objeto a medir y una previa calibracion, generar las medidas requeridas.

Se contempla desarrollar un método que involucre varios métodos ya existentes para facilitar y mejorar la exactitud de las medidas.
